# LinkedIn Campaigns

LinkedIn Campaigns automate prospect discovery and connection requests on LinkedIn. Find your ideal customers and start conversations at scale.

## Overview

LinkedIn Campaigns help you:
- Search for prospects matching your criteria
- Collect profile information automatically
- Send personalized connection requests
- Follow up with new connections
- Track campaign performance

### Campaign Wizard

The wizard guides you through 5 steps:

## Step 1: Campaign Basics

| Field | Required | Description |
|-------|----------|-------------|
| Campaign Name | Yes | Descriptive name for the campaign |
| Description | No | Notes about campaign purpose |

**Example Names:**
- "Tech Founders San Francisco Q1"
- "Marketing Directors Enterprise"
- "SaaS Companies Series A"

## Step 2: Target Audience

Define who you want to reach:

### Search Criteria

| Filter | Description |
|--------|-------------|
| Keywords | Search terms for profiles |
| Location | Geographic targeting |
| Industries | Industry sectors |
| Job Titles | Roles/positions to target |
| Companies | Specific companies |
| Company Size | Employee count range |

### Tips for Targeting

- **Be specific**: Narrow targeting = higher quality
- **Use multiple filters**: Combine criteria for precision
- **Check search volume**: Ensure enough prospects exist

## Step 3: LinkedIn Account

Select which LinkedIn account to use:

| Consideration | Recommendation |
|---------------|----------------|
| Account Health | Choose accounts with high scores |
| Daily Limits | Check available capacity |
| Account Type | Premium/Sales Navigator have better search |

::: warning
The campaign will use this account's daily limits. Monitor usage carefully.
:::

## Step 4: Messages

Configure your outreach messages:

### Connection Request

| Field | Limit | Tips |
|-------|-------|------|
| Connection Message | 300 chars | Short, personalized, value-focused |

**Good Example:**
```
Hi {{name}}, I noticed you're leading growth at {{company}}.
I help similar companies automate their lead generation.
Would love to connect and share some insights. - [Your Name]
```

**Avoid:**
- Generic templates
- Immediate sales pitches
- Long paragraphs
- Spammy language

### Follow-up Message (Optional)

Sent after connection is accepted:

| Field | Description |
|-------|-------------|
| Follow-up Message | Thank them, provide value, suggest next step |
| Delay | Days after acceptance (1-7 recommended) |

**Follow-up Example:**
```
Thanks for connecting, {{name}}!

I saw {{company}} has been growing fast. We recently helped
[similar company] increase their qualified leads by 40%.

Would you be open to a 15-min call to explore if we could
help {{company}} too?
```

## Step 5: Review & Launch

Review all settings before launching:

| Section | What to Check |
|---------|---------------|
| Target Audience | Criteria matches intent |
| Account | Has capacity and good health |
| Messages | Personalized and compelling |
| Limits | Within daily limits |

### Collection Settings

| Setting | Description |
|---------|-------------|
| Daily Limit | Max profiles per day |
| Total Limit | Max profiles for campaign |
| Schedule | When to run (business hours recommended) |

## Campaign Execution

### Collection Phase

The campaign collects profiles matching your criteria:

1. Search runs automatically
2. Profiles are collected and stored
3. Progress updates in real-time
4. Continues until limit reached or no more results

### Outreach Phase

Connection requests are sent:

1. Messages personalized per lead
2. Sent within daily limits
3. Paced throughout the day
4. Tracks acceptance status

### Response Handling

When connections reply:
- Messages appear in Conversations
- AI agents can respond (if configured)
- Leads are updated with interaction

## Best Practices

### Targeting
- Research your ideal customer profile
- Use Sales Navigator for better filters
- Test different criteria combinations
- Focus on quality over quantity

### Messaging
- Personalize every message
- Lead with value, not sales
- Reference something specific
- Keep it short and genuine

### Timing
- Run during business hours
- Avoid weekends (optional)
- Space out requests
- Be patient with responses

### Monitoring
- Check campaign daily
- Review acceptance rates
- Adjust messages if needed
- Pause underperforming campaigns

## Troubleshooting

### Low acceptance rate
- Improve personalization
- Make message less salesy
- Verify targeting is accurate
- Check account health

### Campaign stuck
- Verify account is connected
- Check daily limit availability
- Review for errors
- Try pausing and resuming

### No profiles found
- Broaden search criteria
- Check LinkedIn account access
- Verify filters aren't too narrow
- Try different keywords

### Messages not sending
- Check account daily limit
- Verify account health
- Review campaign status
- Contact support if persists
