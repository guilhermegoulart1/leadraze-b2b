# Campaigns

Campaigns are automated outreach sequences that help you reach leads at scale. GetRaze offers two types of campaigns for different use cases.

## Campaign Types

| Type | Channel | Purpose | Best For |
|------|---------|---------|----------|
| [LinkedIn Campaigns](./linkedin-campaigns) | LinkedIn | Profile collection and connection requests | B2B prospecting, professional networking |
| [Activation Campaigns](./activation-campaigns) | Email, WhatsApp, LinkedIn | Outreach to existing contacts | Follow-up, nurturing, mass outreach |

## Campaign Lifecycle

```
Draft → Scheduled → Active → Paused/Resumed → Completed
```

### Status Descriptions

| Status | Description |
|--------|-------------|
| Draft | Being created, not yet started |
| Scheduled | Set to start at future date |
| Active | Currently running |
| Paused | Temporarily stopped |
| Completed | Finished all actions |
| Failed | Error occurred |

## Quick Start

### LinkedIn Campaign

1. Go to **LinkedIn** → **Campaigns**
2. Click **Create Campaign**
3. Define search criteria
4. Configure messages
5. Review and launch

### Activation Campaign

1. Go to **Activation** → **Campaigns**
2. Click **Create Campaign**
3. Select a Contact List
4. Choose channel and agent
5. Schedule and launch

## Campaign Dashboard

Both campaign types show:

| Element | Description |
|---------|-------------|
| Campaign Card | Name, status, progress |
| Statistics | Leads added, responses, conversions |
| Progress Bar | Completion percentage |
| Actions | Pause, resume, edit, delete |

## Key Metrics

Track campaign performance:

| Metric | Description |
|--------|-------------|
| Leads Added | Profiles collected/contacted |
| Messages Sent | Outreach messages delivered |
| Responses | Replies received |
| Qualified | Leads that converted |
| Response Rate | Responses / Messages |
| Conversion Rate | Qualified / Responses |

## Campaign Best Practices

### Planning
- Define clear objectives
- Know your target audience
- Prepare compelling messages
- Set realistic expectations

### Execution
- Start with smaller batches
- Monitor early results
- Adjust based on feedback
- Respect daily limits

### Optimization
- A/B test messages
- Analyze response patterns
- Refine targeting
- Iterate continuously

## Choosing Campaign Type

| Need | Use |
|------|-----|
| Find new prospects on LinkedIn | LinkedIn Campaign |
| Reach contacts you already have | Activation Campaign |
| Collect business data from Google | Google Maps Agent (not a campaign) |
| Multi-channel outreach | Activation Campaign |
| Connection requests | LinkedIn Campaign |
