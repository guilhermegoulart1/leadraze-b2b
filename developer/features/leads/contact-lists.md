# Contact Lists

Contact Lists help you organize leads into groups for targeted campaigns and better segmentation.

## Overview

Use Contact Lists to:
- Group leads by criteria
- Target specific segments in campaigns
- Organize imported contacts
- Track campaign audiences

## Accessing Contact Lists

Navigate to **Activation** â†’ **Contact Lists** in the sidebar.

## Creating a List

### Manual Creation

1. Click **Create List**
2. Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | List name (e.g., "Tech Companies NYC") |
| Description | No | What this list contains |

3. Click **Create**

### From CSV Import

1. Click **Import CSV**
2. Select or create a list
3. Upload and map your CSV
4. Import contacts

## List Dashboard

The dashboard shows all your lists with:

| Column | Description |
|--------|-------------|
| Name | List name |
| Description | List description |
| Contacts | Number of contacts in list |
| Created | When list was created |
| Actions | Edit, view, delete options |

## Managing List Items

### Viewing Contacts

1. Click on a list name
2. Or click the "View" icon
3. See all contacts in the list

### Adding Contacts

**Add manually:**
1. Open the list
2. Click **Add Contact**
3. Search for existing leads
4. Select and add

**Add from import:**
1. Click **Import to List**
2. Upload CSV
3. Map columns
4. Import

**Add from search:**
1. Perform a lead search
2. Select leads
3. Click **Add to List**
4. Choose destination list

### Removing Contacts

1. Open the list
2. Select contacts to remove
3. Click **Remove Selected**
4. Confirm removal

::: info
Removing from a list doesn't delete the lead from your database.
:::

## Using Lists in Campaigns

### In Activation Campaigns

1. Create an Activation Campaign
2. Select a Contact List as target
3. Configure campaign settings
4. All list contacts receive outreach

### In Google Maps Agents

1. Configure Google Maps Agent
2. Enable activation
3. Collected leads are added to CRM
4. Optionally add to a specific list

## List Best Practices

### Naming Convention

Use descriptive, consistent names:
- `Industry - Location - Qualifier`
- Example: `Tech - NYC - Enterprise`

### Organization Strategies

| Strategy | Use Case |
|----------|----------|
| By Industry | Segment by sector |
| By Location | Geographic targeting |
| By Campaign | Track campaign audiences |
| By Source | Organize by origin |
| By Stage | Funnel-stage lists |

### Size Recommendations

| List Size | Best For |
|-----------|----------|
| < 100 | Highly targeted campaigns |
| 100-500 | Standard campaigns |
| 500-1000 | Larger outreach |
| > 1000 | Mass campaigns (use carefully) |

## List Statistics

Each list tracks:

| Metric | Description |
|--------|-------------|
| Total Contacts | Number of contacts |
| With Email | Contacts with email addresses |
| With Phone | Contacts with phone numbers |
| Campaign Uses | Times used in campaigns |

## Exporting Lists

Export a list to CSV:

1. Open the list
2. Click **Export**
3. Download CSV file

Exported data includes:
- All contact fields
- List membership
- Contact metadata

## Merging Lists

Combine multiple lists:

1. Select lists to merge
2. Click **Merge Lists**
3. Name the new list
4. Choose handling for duplicates:
   - Skip duplicates
   - Keep all

## Splitting Lists

Divide a list:

1. Open the list
2. Click **Split List**
3. Choose split criteria:
   - Random split (50/50, 70/30, etc.)
   - By field value
   - By alphabet

## Troubleshooting

### Contacts not appearing
- Verify import completed
- Check for duplicates
- Refresh the page
- Review import errors

### Can't delete list
- Remove from active campaigns first
- Complete or pause campaigns using the list

### List shows wrong count
- Refresh the page
- Check for recent imports
- Verify no bulk operations in progress
