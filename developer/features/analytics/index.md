# Analytics

Analytics provides insights into your lead generation performance across all channels and campaigns.

## Overview

The Analytics dashboard shows:
- Overall performance metrics
- Campaign performance
- Channel effectiveness
- AI agent statistics
- Lead conversion funnel
- Time-based trends

## Accessing Analytics

Navigate to **Insights** → **Analytics** in the sidebar.

## Dashboard Overview

### Time Period Selection

## Key Metrics Explained

### Response Rate

```
Response Rate = (Responses / Messages Sent) × 100
```

**Benchmarks:**
- LinkedIn: 15-30%
- Email: 5-15%
- WhatsApp: 20-40%

### Conversion Rate

```
Conversion Rate = (Qualified Leads / Total Leads) × 100
```

**Benchmarks:**
- Good: >10%
- Average: 5-10%
- Needs work: <5%

### AI Resolution Rate

```
Resolution Rate = (Resolved by AI / Total AI Conversations) × 100
```

**Benchmarks:**
- Excellent: >80%
- Good: 60-80%
- Needs improvement: <60%
